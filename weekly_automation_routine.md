## DAILY ROUTINE (1 Hour/Day)

### MONDAY — Search & Shortlist (20 min)
```
1. Check Google Alerts email — scan new jobs
2. Search LinkedIn: "Senior Java Developer" Germany (filter: last 24hrs)
3. Search StepStone.de: "Java Lead" or "Java Architect"
4. Add 5 best matches to Job Tracker (job_tracker_template.md)
```

### TUESDAY — Apply (25 min)
```
1. Open Job Tracker — pick 5 jobs from "Not Applied" column
2. For each job:
   a. Open application_templates.md — pick cover letter template
   b. Personalize [brackets] — company name, role, city
   c. Submit application
   d. Update tracker status → "Applied" + date
```

### WEDNESDAY — Recruiter Outreach (20 min)
```
1. Search LinkedIn: "Java recruiter Germany" or "IT recruiter Munich"
2. Find 10 active tech recruiters in Germany
3. Copy-paste Template #3 from application_templates.md
4. Personalize [First Name] and [Company]
5. Send connection request + message
6. Log in tracker: Recruiter Name | Company | Date Contacted
```

### THURSDAY — Follow-Ups (15 min)
```
1. Open Job Tracker
2. Find all "Applied" entries that are 7+ days old with no response
3. Send follow-up using Template #7 from application_templates.md
4. Update status in tracker
```

### FRIDAY — Review & Prep (15 min)
```
1. Count week's activity:
   - Applications submitted
   - Recruiters messaged
   - Responses received
2. Research any companies that replied
3. Prepare for any scheduled calls next week
4. Update Job Tracker status for all active items
```
