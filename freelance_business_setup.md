## SECTION 6 — PAYMENT SETUP (INTERNATIONAL)

### Wise (Best for EUR/GBP/USD from Europe) ✅ RECOMMENDED
- **Setup:** wise.com → Personal account → Add Indian bank account
- **Get:** USD, EUR, GBP virtual account numbers (free)
- **Fee:** 0.4–1.2% conversion (best rates available)
- **Use for:** All European clients, German companies, Irish clients
- **Time to receive:** 1–2 business days

### Payoneer (Best for Upwork/Freelancer payouts) ✅ RECOMMENDED
- **Setup:** payoneer.com → Link to Upwork/Freelancer account
- **Get:** USD receiving account (free)
- **Fee:** $3 per withdrawal to Indian bank OR 2% currency conversion
- **Use for:** All Upwork and Freelancer.com earnings

### PayPal
- **Use for:** Smaller international clients (<$500 transactions)
- **Fee:** 4–5% (expensive for large amounts)
- **Caution:** PayPal can hold funds for new accounts up to 21 days

### UPI / Indian Bank
- **Use for:** Indian clients only
- **Fee:** Free
- **Limit:** Only works within India

### Tax in India — Freelance Foreign Income
- Foreign income earned in India = taxable as professional income
- File ITR-3 or ITR-4 (business/profession income)
- Claim expenses: laptop, internet, software subscriptions
- GST: If annual turnover > ₹20 lakhs → register for GST
- Under ₹20 lakhs: no GST registration needed
- **Consult a CA** for LUT (Letter of Undertaking) to export services without GST

## SECTION 7 — WORK DELIVERY PROCESS

### Delivery Checklist for Every Java Project
- [ ] Code pushed to private GitHub repo (share access to client)
- [ ] README.md written: setup instructions, dependencies, how to run
- [ ] All functions/classes commented (Javadoc for public APIs)
- [ ] Unit tests included (JUnit/Mockito) — minimum 70% coverage
- [ ] .env.example file included (never share real credentials)
- [ ] Postman collection included for all API endpoints
- [ ] Final summary message to client (what was built, what was not in scope, known limitations)

### Delivery Message Template
```
Hi [Name],

Your project is complete. Here is a summary:

✅ Delivered:
- [List what was built]
- [GitHub repo link]
- [Postman collection attached]

📋 How to run:
- [2-3 line quick start]

⚠️ Notes:
- [Any known limitations or out-of-scope items]

Please review and let me know if you have any questions or need adjustments.
If everything is satisfactory, I would really appreciate a review on [platform].

It was a pleasure working with you. I am available for follow-up work anytime.

Mayank
```

### How to Get 5-Star Reviews
1. Deliver 1–2 days BEFORE deadline (underpromise, overdeliver)
2. Send a mid-project update even if not asked
3. Document your code clearly
4. Ask 1–2 clarifying questions early (shows professionalism)
5. After delivery, ask: "Is there anything else I can clarify or improve?"
6. Ask for review ONLY after client confirms satisfaction
